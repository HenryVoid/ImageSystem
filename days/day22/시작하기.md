# 22일차: Vision 얼굴·텍스트 인식

## 학습 목표

- Vision 프레임워크의 기본 동작 원리(Request, Handler, Observation) 이해
- `VNDetectFaceRectanglesRequest`를 이용한 얼굴 위치 감지 구현
- `VNRecognizeTextRequest`를 이용한 OCR(광학 문자 인식) 구현
- 여러 Vision 요청을 동시에 처리하는 파이프라인 구성

## 실습 내용

1. **VisionManager 구현**: Vision 요청을 처리하는 핵심 로직 작성
2. **FaceDetectionView**: 사진을 선택하고 얼굴을 찾아 빨간 박스로 표시하기
3. **TextRecognitionView**: 사진 속 텍스트를 읽어내고 화면에 표시하기
4. **CombinedVisionView**: 얼굴과 텍스트를 한 번에 찾기

## 핵심 파일

- `VisionManager.swift`: Vision 요청 처리 로직
- `FaceDetectionView.swift`: 얼굴 인식 UI
- `TextRecognitionView.swift`: 텍스트 인식 UI
