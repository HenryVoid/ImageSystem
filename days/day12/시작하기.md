## 🆘 문제 해결

### 이미지가 로드되지 않음

**원인**: 네트워크 연결 없음

**해결**:
```
1. Wi-Fi 연결 확인
2. 시뮬레이터 재시작
3. Xcode Clean Build (⇧⌘K)
```

### 앱이 느림

**원인**: Debug 모드

**해결**:
```
Release 모드로 실행:
1. Product → Scheme → Edit Scheme
2. Run → Build Configuration → Release
3. Close → ⌘R로 재실행
```

### 메모리 경고

**원인**: 1000개 이미지 + 큰 크기

**해결**:
```
1. 이미지 개수를 100개로 변경
2. 크기를 "작게" 또는 "중간"으로 변경
3. 시뮬레이터 재시작
```
